# 📊 Super Store Sales Analysis & Dashboard using Python / Power BI

This project presents a comprehensive **Sales Analysis and Interactive Dashboard** built on Super Store data. It uncovers key business insights related to **sales, profit, customer segments, categories, and regional performance**.

---

## 📌 Project Objective

- Analyze sales and profit trends over time  
- Understand customer segments and purchasing behavior  
- Identify top-performing categories and sub-categories  
- Evaluate regional and shipping performance  
- Build an interactive dashboard for business insights  

---

## 🧰 Tools & Technologies Used

- **Python**
  - Pandas, NumPy – data manipulation  
  - Matplotlib, Seaborn – data visualization  
- **Power BI** – dashboard creation  
- **Jupyter Notebook / Google Colab** – development  
- **GitHub** – version control  

---

## 📂 Dataset Overview

The dataset contains retail sales data with:

- Order Date, Ship Date  
- Customer details (ID, Name, Segment)  
- Location (Country, State, Region)  
- Product Category & Sub-Category  
- Sales, Quantity, Profit  
- Ship Mode  

---

## 📊 Data Analysis Steps

### 1️⃣ Data Cleaning & Preprocessing
- Converted date columns into datetime format  
- Removed unnecessary columns  
- Created Month column for trend analysis  
- Checked missing values  

---

### 2️⃣ Exploratory Data Analysis (EDA)

- Monthly Sales Trend  
- Category-wise Sales Distribution  
- Average Sales by Category  
- Sales vs Profit Scatter Plot  
- Sub-category Analysis  
- Segment-wise Sales  
- Shipping Mode Analysis  
- Region-wise Performance  

---

### 3️⃣ Dashboard Features

- **KPIs**: Total Sales, Profit, Quantity  
- **Monthly Sales Trend (YoY)**  
- **Sales by Category & Sub-category**  
- **Segment-wise Analysis (Consumer, Corporate, Home Office)**  
- **Payment Mode Distribution**  
- **Region-wise Sales & Profit**  
- Interactive Filters (Central, East, South, West)  

---

## 📊 Key Insights

- Office Supplies has the highest total sales  
- Technology has the highest average sales (high-value products)  
- Consumer segment contributes the most sales  
- Standard shipping is most commonly used  
- Sales show fluctuations with peaks during certain months  
- Some transactions result in losses  

---

## 💡 Business Recommendations

- Focus on Technology category for higher profit  
- Reduce loss-making transactions  
- Improve sales in low-performing regions  
- Promote high-performing sub-categories  
- Optimize shipping strategy
