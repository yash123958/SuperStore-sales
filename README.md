Super Store Sales Analysis & Dashboard using Python / Power BI

This project presents a comprehensive Sales Analysis and Interactive Dashboard built on Super Store data. It uncovers key business insights related to sales, profit, customer segments, categories, and regional performance. The goal is to help businesses make data-driven decisions to improve revenue and profitability.

📌 Project Objective
Analyze sales and profit trends over time
Understand customer segments and purchasing behavior
Identify top-performing categories and sub-categories
Evaluate regional and shipping performance
Build an interactive dashboard for business insights
🧰 Tools & Technologies Used
Python
Pandas, NumPy → data cleaning & analysis
Matplotlib, Seaborn → visualization
Power BI → dashboard creation
Google Colab / Jupyter Notebook → development
GitHub → version control
📂 Dataset Overview

The dataset contains sales records of a retail superstore, including:

Order Date, Ship Date
Customer ID, Customer Name, Segment
Country, State, Region
Product Category & Sub-Category
Sales, Quantity, Profit
Ship Mode
📊 Data Analysis Steps
1️⃣ Data Cleaning & Preprocessing
Converted Order Date to datetime format
Removed unnecessary columns (like empty fields)
Created Month-wise data for trend analysis
Checked missing values and ensured data consistency
2️⃣ Exploratory Data Analysis (EDA)
Monthly Sales Trend 📈
Category-wise Sales Distribution 📊
Average Sales by Category
Sales vs Profit (Scatter Plot)
Sub-category performance (Top products)
Segment-wise sales comparison
Shipping mode analysis
Regional performance
3️⃣ Dashboard Features
KPIs: Total Sales, Profit, Quantity
Sales Trend (YoY)
Category & Sub-category Analysis
Segment-wise Sales (Consumer, Corporate, Home Office)
Payment Mode Distribution
Region-wise Sales & Profit
Interactive Filters (Region: Central, East, South, West)
📊 Key Insights
Office Supplies contributes the highest total sales, but low average value
Technology has higher average sales → more profitable per order
Consumer segment dominates overall sales
Standard shipping mode is most used
Sales show seasonal spikes (especially towards year-end)
Some transactions show loss even at high sales values
💡 Business Recommendations
Focus on Technology category to increase profit margins
Reduce losses by analyzing discount-heavy transactions
Improve performance in low-performing regions (e.g., South)
Promote high-performing sub-categories (Phones, Chairs, Binders)
Optimize shipping strategies to reduce cost
🧠 Concepts Covered
Data Cleaning & Preprocessing
Time Series Analysis

Data Visualization (Line, Bar, Pie, Scatter)
Dashboard Design
Business Insight Generation
✅ Conclusion

This project provides a complete analysis of retail sales data and presents insights through an interactive dashboard. It helps understand sales patterns, customer behavior, and profitability drivers, enabling better strategic decisions.
